# Problème #3

Vous devez créer un plugin pour l'éditeur de texte Vim, permettant de jouer au jeu "Plus ou moins" lorsqu'on entre la commande :PlusOuMoins .

Le jeu "Plus ou moins" est un jeu classique en programmation, dans lequel l'ordinateur sélectionne un nombre aléatoire entre 0 et 1000. L'objectif du joueur est de deviner le nombre. Pour l'aider, l'ordinateur lui fournit des indices sur le nombre à trouver (plus petit que, plus grand que). Le joueur peut essayer tant qu'il n'a pas trouvé le nombre. Lorsqu'il réussit, on lui affiche son nombre d'essai et on lui donne la possibilité de rejouer.

Pour ce problème, vous devez obligatoirement utiliser le langage VimScript.
